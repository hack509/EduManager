Stack Technique

Technologies modernes
et éprouvées

Construit sur des bases solides pour garantir performance, sécurité et évolutivité à long terme. Architecture multi-tenant avec isolation totale des données par établissement.

🐍
Python / Flask 3.1 Backend robuste & rapide
🗃
PostgreSQL + SQLAlchemy 2.x Base de données relationnelle
⚡
HTMX + Alpine.js UI réactive sans framework lourd
🎨
TailwindCSS Design system cohérent
📄
WeasyPrint Génération PDF haute qualité

Sécurité multi-tenant

Isolation totale des données. RBAC granulaire avec rôles et permissions par utilisateur.

100% Responsive
